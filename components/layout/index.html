Layout #

布局容器，可以与d-header, d-content, d-footer, d-aside组合实现布局； d-layout下可嵌套元素：d-header, d-content, d-aside, d-layout。

Header #

顶部布局，只能放在d-layout容器中，作为d-layout容器的顶部实现。 默认高度：40px。

Footer #

底部布局，只能放在d-layout容器中，作为d-layout容器的底部实现。

Content #

内容容器，只能放在d-layout容器中，作为d-layout容器d-header与d-footer之间的内容。

Aside #

侧边栏，只能放在d-layout容器中，作为d-layout容器的侧边栏部分。
