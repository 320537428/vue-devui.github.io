TreeSelect 参数 #

参数类型默认说明跳转
treeDataTreeData必选，树形选择框的内容基本用法
placeholderstring'请选择'可选，修改输入框的默认显示内容基本用法
disabledbooleanfalse可选，值为 true 时禁止用户使用禁用
allowClearbooleanfalse可选，值为 true 时可以清空输入框内容可清空
multiplebooleanfalse可选，值为 true 时可选择多个项多选
leafOnlybooleanfalse可选，值为 true 时仅可选择叶子节点仅叶子节点可选
enableLabelizationbooleanfalse可选，值为 true 时仅可选择叶子节点标签化

接口 & 类型定义 #

interface TreeItem {
  id: number | string
  label: string
  parent?: TreeItem
  children?: Array<TreeItem>
  level?: number
  loading?: boolean
  opened?: boolean
  checked?: boolean
  halfchecked?: boolean
  disabled?: boolean
  
  [prop: string]: any
}

type TreeData = Array<TreeItem>
